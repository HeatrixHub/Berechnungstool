# Legacy Tkinter UI (Archiv)

Dieser Ordner enthält die frühere Tkinter-basierte Oberfläche und alle
zugehörigen Plugin- und Tab-Implementierungen. Der Code wird nur als Referenz
aufbewahrt und ist **kein** Bestandteil der aktiven Anwendung.

Die produktive UI befindet sich unter `app/ui_qt/` und wird über
`app/main_qt.py` gestartet.
